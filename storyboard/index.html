🗄️ Database ERD (4 Tables)

Table Name Primary Key Foreign Key Description
users user_id - Student and admin login information
facilities facility_id - Sports courts (badminton, futsal, basketball)
equipment equipment_id facility_id Sports equipment with status tracking
bookings booking_id user_id, facility_id Time slot bookings by students

📊 Equipment Status Options:

  • ✅ Available - Ready for rental
  • ⚠️ Damaged - Needs repair, not available
  • 🔧 In-Maintenance - Currently being serviced

🔗 Table Relationships:

  • 👤 users (1) -----< (N) bookings (One user can make many bookings)
  • 🏟️ facilities (1) -----< (N) bookings (One facility can have many bookings)
  • 🏸 facilities (1) -----< (N) equipment (One facility can have many equipment)
